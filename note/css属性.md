
# 字体、文本

- font-size：字体大小
- color：文本颜色
- text-align：对齐方式
- line-height：行高

# 背景

background：背景，复合属性

# 边框

border：设置边框，复合属性

# 尺寸

- width：宽度
- height：高度

# 盒子模型：控制布局

margin：外边距
padding：内边距
注意：默认情况下内边距会影响整个盒子的大小。设置box-sizing: border-box;设置盒子的属性，让width和height就是最终盒子的大小。
float：浮动
分类
left：左浮动
right：右浮动
